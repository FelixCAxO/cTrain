# cTrain Language Tracks

cTrain separates languages by learning goal, not just syntax.

| Track | Focus | Best use |
|---|---|---|
| C | Systems foundations | Memory, pointers, structs, compilation, debugging |
| Java | Certification and enterprise Java | Oracle Java SE 25 preparation and modern Java fluency |
| Python | Scripting, automation, and Python certification | PCEP, PCAP, and PCPP1 progression |

Java is the certification-heavy track.
Python is the certification-ladder and scripting-productivity track.
C is the systems-understanding track.

## Python Certification Ladder

| Level | Python cert | Best for |
|---|---|---|
| Entry | PCEP - Certified Entry-Level Python Programmer | Python basics |
| Associate | PCAP - Certified Associate Python Programmer | Solid junior Python proof |
| Professional | PCPP1 - Certified Professional Python Programmer 1 | Advanced Python/OOP/modules/tools |
